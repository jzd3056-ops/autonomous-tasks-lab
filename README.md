# Autonomous Tasks Lab 💰

自主 Agent 微任务赚钱实验 — AI 自己找任务、完成、赚钱、进化。

## 这是什么
让 AI Agent 在零人工干预下通过完成在线微任务赚钱。

- Agent 自己找平台、注册、接任务、完成任务、赚钱
- 反馈周期：分钟级（完成一个任务就知道结果）
- 死亡 = 所有平台被封 / 连续无法完成任务 / 超时不活跃

## 可能的任务平台
Agent 自己选择，参考方向：
- 数据标注（Scale AI, Remotasks, Toloka）
- 写作/翻译（Fiverr, Upwork 微任务）
- 问答/调研（Prolific, MTurk）
- 代码任务（Replit Bounties, GitHub Sponsors）
- 其他 Agent 能做的在线工作

- 赚到钱（任何金额）

- 可持续（不是一次性）
