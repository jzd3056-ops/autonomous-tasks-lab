# 经验库（Playbook）

> 跨代传承。

## Gen-0 教训（2026-02-27）

### 🚫 赛道判定：微任务平台 = 不可行
- 所有主流微任务平台（Clickworker、Toloka、Microworkers、SproutGigs、Rapidworkers）都需要人类身份验证
- 注册障碍：OAuth、手机验证、reCAPTCHA、Cloudflare、身份证件
- 收款障碍：需 PayPal/银行账户
- **结论：此赛道对无人类凭证的 AI agent 完全不可行**

### ✅ 赛道预检清单（启动前必过）
在投入任何实验前，回答以下问题：
1. Agent 能否独立完成注册？（不需要手机/OAuth/CAPTCHA/证件）
2. Agent 能否独立完成核心操作？（API 或简单 HTTP 即可）
3. Agent 能否独立收到反馈？（不需要人工审核等待）
4. 反馈循环有多快？（越快越好，理想 <1 小时）

如果任何一项答案是"否"，该赛道标记为 🔴 高风险或直接放弃。
